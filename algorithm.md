### **一、斐波那契数列**

**1、暴力递归**

斐波那契数列的数学形式就是递归的，写成代码就是这样：

```text
int fib(int N) {
    if (N == 1 || N == 2) return 1;
    return fib(N - 1) + fib(N - 2);
}
```

**递归算法的时间复杂度：子问题个数乘以解决一个子问题需要的时间**

即 O(2^n)，指数级别，爆炸。
